# Changelog

## v0.1.0 - 2026-05-27

Initial public release of Room Layout AI.

### Added

- Spatial AI Playground concept and GitHub Pages demo.
- 2D room editor with adjustable room width and depth.
- Furniture palette for bed, desk, chair, sofa, monitor, and shelf.
- Drag, select, rotate, and remove furniture interactions.
- Heuristic Optimize flow with animated furniture movement.
- Door clearance, collision avoidance, wall alignment, and open-path scoring.
- Animated score cards for Walking Efficiency, Workspace Score, Comfort Score, Free Space, and Focus Score.
- Before/after movement trails and score deltas after optimization.
- AI suggestion panel and copyable layout summary.
- GitHub Actions workflow for GitHub Pages deployment.
